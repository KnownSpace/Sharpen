#### Requires:
  - Windows version >= 7 or Linux kernel version >= 2.6.28
  - Cmake version >= 3.14
  - Msvc >= 19.0 or Clang >= 3.5 or GCC >= 4.8

#### To build:
##### Just run `build.sh` or `build.cmd`

#### Use in your projects:
  1. Build Sharpen
  1. Add include directory
  1. Add link target
